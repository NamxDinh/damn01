# Bài thực hành 5 – PHP

## Câu 1: Vẽ tam giác và hình chữ nhật bằng dấu `*`

```php
<?php
// Tam giác
$n = 5;
echo "Tam giac:<br>";
for($i = 1; $i <= $n; $i++){
    for($j = 1; $j <= $i; $j++){
        echo "* ";
    }
    echo "<br>";
}

echo "<br>";

// Hình chữ nhật
$rows = 4;
$cols = 6;
echo "Hinh chu nhat:<br>";
for($i = 1; $i <= $rows; $i++){
    for($j = 1; $j <= $cols; $j++){
        echo "* ";
    }
    echo "<br>";
}
?>
```

# Câu 3: Tính tổng các số lẻ từ 1 đến n

```php
<?php
$n = 10;
$sum = 0;

for($i = 1; $i <= $n; $i++){
    if($i % 2 != 0){
        $sum += $i;
    }
}

echo "Tong so le tu 1 den $n la: $sum";
?>
```

# Câu 4: Tính tổng

S = 1/2 + 1/4 + 1/6 + ... + 1/(2*n)

```php
<?php
$n = 5;
$S = 0;

for($i = 1; $i <= $n; $i++){
    $S += 1/(2*$i);
}

echo "Tong S = $S";
?>
```

---

# Câu 5: Tính tổng

S = 1 − 2 + 3 − 4 + ... + (-1)^(n-1) * n

```php
<?php
$n = 5;
$S = 0;

for($i = 1; $i <= $n; $i++){
    $S += pow(-1, $i-1) * $i;
}

echo "Tong S = $S";
?>
```

# Câu 6: Tính giai thừa n (n!)

    $gt *= $i;

# Câu 7: Tính số Fibonacci thứ n

```php
<?php
$n = 10;

$a = 0;
$b = 1;

for($i = 2; $i <= $n; $i++){
    $c = $a + $b;
    $a = $b;
    $b = $c;
}

echo "Fibonacci thu $n la: $b";
?>
```

---

# Câu 8: Tổng các số Fibonacci từ 1 đến n

```php
<?php
$n = 10;

$a = 0;
$b = 1;
$sum = $a + $b;

for($i = 2; $i < $n; $i++){
    $c = $a + $b;
    $sum += $c;
    $a = $b;
    $b = $c;
}

echo "Tong Fibonacci den n la: $sum";
?>
```
